# 劳务派遣管理平台

基于 B/S 架构的劳务派遣管理系统，采用前后端分离技术，解决劳务派遣中信息不对称、流程繁琐及结算复杂等痛点。

## 技术栈

| 层级 | 技术选型 |
|------|----------|
| 前端 | Vue 3 + element-plus + vite + axios + pinia|
| 后端 | Spring Boot 3 + MyBatis-Plus + satoken|
| 数据库 | MySQL redis|
| 架构 | 前后端分离 RESTful API |

## 用户角色 (RBAC)

- **系统管理员** - 基础数据维护、用户管理、全局监控
- **企业客户** - 发布用工需求、筛选人才、管理考勤与结算
- **派遣员工** - 维护简历、查看应聘进度及薪资明细

## 功能模块

### 1. 资源管理
- 人才库：简历录入/解析、技能标签管理
- 企业库：企业入驻审核、合同归档

### 2. 岗位与匹配
- 岗位需求发布（名称、技能要求、薪资、地点）
- 多维度筛选（技能、地域、薪资区间）
- 智能推荐匹配

### 3. 派遣全流程管理
- 状态流转：面试邀约 → 录用 → 签约 → 入职 → 在职 → 离职/退回
- 面试管理与入职签约
- 在职状态跟踪与异常处理

### 4. 薪酬与结算
- 自定义薪资规则（时薪/日薪/月薪）
- 考勤数据集成
- 自动核算（员工薪资 + 企业账单）
- 可视化报表与导出

## 项目结构

```
├── frontend/          # Vue 3 前端项目
└── backend/           # Spring Boot 3 后端项目
```

## 快速开始

### 后端
```bash
cd backend
mvn spring-boot:run
```

### 前端
```bash
cd frontend
npm install
npm run dev
```
